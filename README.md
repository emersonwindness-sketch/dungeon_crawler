# dungeon_crawler
Not something to take seriously, just putting into practice code i learn in my studies.

But to an actual read-me, here's what this is.

Welcome to the most original name you'll ever see since the first video game ever back in 1958 "Tennis for Two": Dungeon Crawler

It has a simple premise of crawling into the darkness of the unknow Dungeon, but at the moment, it does not have the said dungeon, just some mechanics that i'm still working on:

- World building
- Dynamic inventory
- Trading
- Basic combat mechanics

For now, the inventory and the walking around the world ( if you can call it that ) works as i intended, you will type everything to go and do things as well as making any action.
There is no interface, so prepare your fingers ( MAYBE i'll do an basic interface with some actual grapichs in the future )
